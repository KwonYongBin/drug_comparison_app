# Drug Safety Compare

의약품 정보를 검색하고 성분을 비교하여
병용 위험 여부를 확인할 수 있는 웹 서비스입니다.

사용자가 복용 중인 의약품의 성분을 비교하여
중복 성분이나 병용 주의 여부를 확인할 수 있도록 돕는 것을 목표로 합니다.

본 서비스는 의료 판단을 제공하지 않으며
의약품 복용 관련 최종 판단은 반드시 의사 또는 약사와 상담해야 합니다.

# 프로젝트 개요

실제 의약품을 복용할 때
다른 약과 함께 복용해도 되는지 확인하기 어렵다는 문제를 경험했습니다.

특히 감기약, 진통제 등은
여러 제품에 동일한 성분이 포함되는 경우가 많아
중복 복용 위험이 발생할 수 있습니다.

이러한 문제를 해결하기 위해
공공 의약품 데이터를 활용하여
의약품 정보를 검색하고 비교할 수 있는 서비스를 기획했습니다.

---

# 프로젝트 목표

- 공공 의약품 API를 활용한 정보 검색 서비스 구현
- 의약품 성분 비교 기능 제공
- 병용 위험 정보 표시 (DUR 기반)
- 사용자에게 안전한 의약품 정보 탐색 경험 제공

---

# 주요 기능

## 의약품 검색
- 의약품명 또는 성분명으로 검색
- 공공 의약품 데이터 기반 검색 결과 제공
- 검색 결과 최대 10개 표시

## 의약품 정보 카드
- 의약품명
- 일반/전문의약품 구분
- 주요 성분 정보
- 병용 위험 여부 표시

## 의약품 비교 기능
- 최대 2개의 의약품 비교
- 주요 성분 비교
- 중복 성분 여부 확인
- 병용 주의 정보 제공

## 안전 안내
- 공공 의약품 안전사용서비스(DUR) 기준 정보 제공
- 의료 판단이 아닌 참고 정보 제공
- 전문가 상담 권장 안내 표시

# AI 보조 기능 (예정)
